# 🔥 OvenEye – Smart Oven Safety System

**OvenEye** is an ESP32-based IoT safety system designed to monitor oven and stove activity in real time. It uses a **Hall effect sensor** to detect when an oven or stove knob is turned and sends alerts to a mobile app, helping prevent accidents and improve kitchen safety.

This project combines **hardware**, **embedded programming**, and **cloud connectivity** to create a real-world safety solution. OvenEye is built with a focus on **reliability**, **usability**, and **future expandability**.

---

## 🚀 Why OvenEye?

Kitchen accidents often happen when ovens or stoves are left unattended. OvenEye was created to:

- Increase kitchen safety  
- Provide real-time monitoring and alerts  
- Explore IoT and embedded systems using the ESP32  

---

## 🛠️ Technologies Used

- **ESP32** microcontroller  
- **Hall effect sensor**  
- **Blynk** IoT platform  
- **Arduino / C++**
